# ジョブの監視

ジョブは、量子サービスチャネルのformat@@0ページにリストされます。

-   IBM Cloud チャンネル: IBM Cloud コンソールの量子format@@0(https://cloud.ibm.com/quantum/instances)から、インスタンスの名前をクリックし、format@@1タブをクリックします。
-   IBM Quantum channel: IBM Quantum Platformでformat@@0(https://quantum.ibm.com/jobs)を開きます。

ジョブインスタンスを使用して、ジョブのステータスを確認したり、適切なコマンドを呼び出して結果を取得したりします。

|                               |                                                                                                                                                                                                      |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| job.result()                  | ジョブが完了した直後にジョブの結果を確認します。 ジョブの結果は、ジョブが完了した後に表示されます。 そのため、job.result() はジョブが完了するまでブロックされた呼び出しです。                                                                                                       |
| job.job_id()                  | そのジョブを一意に識別する ID を返します。 ジョブの結果を後で取得するには、ジョブ ID が必要です。 したがって、後で取得したいジョブのIDを保存することをお勧めします。                                                                                                             |
| job.status()                  | ジョブの状態を確認します。                                                                                                                                                                                        |
| job = service.job(\<job_id>)  | 以前に送信したジョブを取得します。 この通話にはジョブ ID が必要です。                                                                                                                                                                |

<span id="retrieve-later"></span>
## 後でジョブの結果を取得する

以前に送信したジョブを取得するには、 `service.job(\<job\how_toID>)`を呼び出します。 ジョブIDを持っていない場合、または一度に複数のジョブを取得したい場合は、引退したシステムからのジョブを含め、`serviceを呼び出します。 obs()`にはオプションのフィルタがあります。 [QiskitRuntimeService.jobs](../api/qiskit-ibm-runtime/qiskit_ibm_runtime.QiskitRuntimeService#jobs) を参照してください。

<Admonition type="note">
  service.jobs() は Qiskit Runtime ジョブのみを返します。 他のジョブを取得するには、代わりに [qiskit-ibm-provider](../api/qiskit-ibm-provider/qiskit_ibm_provider.IBMBackend#ibmbackend) を使用してください。
</Admonition>

## 例

この例では、`my_backend` で実行された最新の 10 のランタイムジョブを返します。

```python
from qiskit_ibm_runtime import QiskitRuntimeService

# 最初にアカウントを初期化します。
service = QiskitRuntimeService()

service.jobs(backend_name=my_backend)
```
