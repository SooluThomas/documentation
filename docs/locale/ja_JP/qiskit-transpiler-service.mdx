# Qiskit transpiler サービスを使用したトランスパイル回路の遠隔操作

Qiskitトランスパイラーサービスは、クラウド上でトランスパイル機能を提供します。 ローカルの Qiskit トランスパイラー機能に加えて、トランスパイレーションタスクは、IBM Quantum Cloud リソースと AI 搭載トランスパイラーパスの両方から恩恵を受けることができます。

Qiskit transpiler サービスは、このサービスとその機能を現在の Qiskit パターンとワークフローにシームレスに統合するための Python ライブラリを提供します。

<Admonition type="note">
    この実験サービスは、IBM Quantum Premium Plan ユーザのみが利用できます。
    サービスはアルファリリースであり、変更されることがあります。
</Admonition>

<span id="install-transpiler-service"></span>
## qiskit-transpiler-service パッケージをインストールする

Qiskit transpiler サービスを使用するには、`qiskit-transpiler-service` パッケージをインストールします。

```sh
pip install qiskit-transpiler-service
```

デフォルトでは、パッケージは QisKit API トークンを定義した IBM Quantum サービスへの認証を試みます。 そして、`QISKIT_IBM_TOKEN` 環境変数またはファイル `~/ からトークンを使用します。 iskit/qiskit-ibm.json` (`default-ibm-quantum`のセクションの下)。

## qiskit-transpiler-service transpile オプション

-   `target` (optional, str) - QiskitRuntimeServiceによって期待されるバックエンド名（例えば、 `ibm_sherbrooke`）。 これが設定されている場合、transpile メソッドはトランスパイル操作に指定されたバックエンドからのレイアウトを使用します。 `coupping_map`のような他のオプションが設定されている場合、`target`の設定は上書きされます。
-   `coupping_map` (オプション, List\[List[int]]) - 有効なカップリングマップリスト（例えば\[[0,1],[1,2]]）。 これが設定されている場合、transpile メソッドはトランスパイル操作にこのカップリングマップを使用します。 定義されている場合は、`target` に指定された値を上書きします。
-   `optimization_level` (int) - 移調処理中に適用される潜在的な最適化レベル。 有効な値は[1,2,3]で、1は最小の最適化(そして最も速く)、3は最も最適化(そしてほとんどの時間を要する)です。
-   `ai` (bool) - 移調時にAI機能を使用するかどうか。 利用可能なAI機能は、「AIRouting」トランスパイルパスまたは他のAI合成方法で使用できます。 この値が `True` の場合、要求された `optimization_level` に応じて、異なるAIによるトランスパイルが適用されます。
-   `qiskit_transpile_options` (dict) - \[Qiskit `transpile()` メソッドで有効な他のオプションを含めることができるPython辞書オブジェクト。 `qiskit_transpile_options` 入力に `optimization_level` が含まれている場合は、パラメータ入力として指定された `optimization_level` を優先して破棄されます。 `qiskit_transpile_options` に Qiskit `transpile()` メソッドで認識されないオプションが含まれている場合、ライブラリはエラーを発生させます。

## 例

以下の例では、Qiskitトランスパイラーサービスを使用してさまざまなパラメータを使用して回路を移調する方法を示します。

1.  ランダムな回路を作成し、Qiskit transpiler サービスを呼び出し、`target`として`ibm_cairo`を使用して回路を移調します。 1 は `optimization_level` として、トランスパイル時にはAIを使用しません。

    ```python
    from qiskit.circuit.random import random_circuit
    from qiskit_transpiler_service.transpiler_service import TranspilerService

    random_circ = random_circuit(5, depth=3, seed=42).decompose(reps=3)

    cloud_transpiler_service = TranspilerService(
        target="ibm_cairo",
        ai=False,
        optimization_level=1,
    )
    transpiled_circuit = cloud_transpiler_service.run(random_circ)
    ```

2.  同じようなランダム回路を作成し、トランスパイルします。フラグ「ai」を「True」に設定して、AIのトランスパリング機能を要求します。

    ````
    ```python
    from qiskit.circuit.random import random_circuit
    from qiskit_transpiler_service.transpiler_service import TranspilerService

    random_circ = random_circuit(5, depth=3, seed=42).decompose(reps=3)

    cloud_transpiler_service = TranspilerService(
        target="ibm_cairo",
        ai=True,
        optimization_level=1,
    )
    transpiled_circuit = cloud_transpiler_service.run(random_circ)
    ```
    ````
