# ジョブの実行時間を推定する

IBM Quantum チャンネルにジョブを送信した後 \`job を使ってジョブを実行するのにどれだけの_量子時間_がかかるか推定できます。 sage_estimation  別の方法として、[IBM Quantum Platformのユーザーインターフェースでこの情報を表示する](#view-usage) ができます。

量子時間とは、長さ、秒単位で、量子システムは、ユーザーの要求を満たすことを約束します。

<Admonition type="note" title="Notes">
    - これはプリミティブを使用するジョブにのみ適用されます。
    - クラウドチャンネルの IBM Qiskit Runtime ではまだ利用できません。 
</Admonition>

例

```python
from qiskit import QuantumCircuit
from qiskit_ibm_runtime import QiskitRuntimeService, Sampler

service = QiskitRuntimeService()

# Create a new circuit with two qubits (first argument) and two classical
# bits (second argument)
qc = QuantumCircuit(2, 2)

# Add a Hadamard gate to qubit 0
qc.h(0)

# Perform a controlled-X gate on qubit 1, controlled by qubit 0
qc.cx(0, 1)

# Measure qubit 0 to cbit 0, and qubit 1 to cbit 1
qc.measure(0, 0)
qc.measure(1, 1)

# Run on the least-busy backend you have access to
backend = service.least_busy(simulator=False,operational=True)

# Create a Sampler object
sampler = Sampler(backend)

# Submit the circuit to the sampler
job = sampler.run(qc)

print(job.usage_estimation)
```

出力:

```python
{'quantum_seconds': 4.1058720028432445}
```

## IBM Quantum Platformでの推定ジョブ使用状況を見る

IBM Quantum Platformでは、推定使用量(ジョブが実行されるまでにどのくらいの時間がかかるか)を2箇所で確認できます。

-   format@@0(https://quantum.ibm.com/jobs)のformat@@1で。  ホームページから、「最近のジョブ」テーブルの「_すべて表示_」をクリックします。  format@@0列には、保留中のジョブの推定使用量または完了済みジョブの実際の使用量が表示されます。 
-   仕事の詳細ページで。 [Dashboard](https://quantum.ibm.com/) または format@@1(https://quantum.ibm.com/jobs) から、ジョブIDをクリックしてジョブの詳細ページを開きます。  推定使用量はformat@@0に表示されます。 

## 次のステップ

<Admonition type="tip" title="Recommendations">
    - 以下のヒントを確認してください: format@@2(minimize-time)
    - format@@0(max-execution-time) を設定します。
</Admonition>
