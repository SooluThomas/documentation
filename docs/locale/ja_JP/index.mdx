# はじめに

IBM Quantum は、7 つの [utility-scale](https://www.ibm.com/blog/announcement/new-ibm-quantum-systems-on-the-ibm-cloud/) 量子系の世界最先端の艦隊を維持しています。 これらのシステムは比類のない信頼性を示し、量子システムの艦隊全体で95%以上の稼働時間があります。比類のない安定性です。 2量子ビットゲートの誤差変動は0以下です。 01は月単位で測定された時間スケール[^1]。

## ランフェーズ

実行フェーズでは、量子プログラムを量子システム上で実行するように送信します。 このセクションでは、IBM Quantum ハードウェアに関する情報と、量子システムへのアクセスを提供するインスタンスへの接続方法について説明します。 ジョブの実行時間とコストの推定、セッション内での実行、システム上での時間の予約などについての詳細を見ることができます。 

実行フェーズ中のステップは次のとおりです。

1.  アカウントの資格情報を使用して、選択したチャネルを認証します ([IBM Quantum Platform](../start/setup-channel#ibm-quantum-platform)または[IBM Cloud](../start/setup-channel#ibm-cloud))。
2.  バックエンドを選択
3.  ジョブを量子システムに送信します。
4.  ジョブの結果を表示

[^1]: 2023年7月20日から9月20日まで、アクセス可能なすべてのイーグルプロセッサで測定された2Qゲートエラー。
