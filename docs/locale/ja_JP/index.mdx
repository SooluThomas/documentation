# はじめに

量子コミュニティは、2016年からクラウド上で量子コンピューティングを研究しています。  今度はコミュニティに新しいチャレンジがあります:_有用な量子計算を実証_。 

IBM Quantumを通じて、あなたの仕事のためのパフォーマンスとユーティリティスケール(>100-qubit)の量子システム、並びにスケーラブルで柔軟な量子ソフトウェアにアクセスすることができます。 無償プランを試したり、有料プランやプレミアムプランを使ってプログラムを実行したりするかどうか 量子コンピューティングで役に立つ研究ができるようになりました

このドキュメントの核となるセクションは、一般的な量子ユーザーの旅の段階に基づいています。Qiskitで量子回路と演算子を構築します。 そして、QisKit Runtime プリミティブを使用して量子システム上でトランスパイルを実行し、回路実行のための簡素化されたインターフェイスを使用します。 Qiskit Runtime とペアリングすることで、高度なランタイムコンパイル、エラー抑制、およびエラー軽減の手法によって、シームレスな回路実行が可能になります。 

IBM Quantumを使い始めたばかりの方は、適切な場所にいらっしゃいます。 このセクションのトピックを使用して、準備をしてから、 [Build](../build) に進み、最初の量子回路を作成します。 このページの上部にあるタブに従って、セクションを順番に作業し続けます。

お探しの内容は既にご存知ですか？

-   [Install](インストール): インストールとセットアップの手順でツールとプラットフォームの使用を開始します。

-   [Build](../build): 動的回路や中間回路測定などの高度な手法やプリミティブを用いた量子回路の設計・開発。 回路ライブラリはこちらもご覧いただけます。

-   [Transpile](../transpile): エラー認識度が異なり、ハードウェア上で効率的に動作するように回路を最適化するようにコンパイルします。

-   [Verify](../verify): 量子回路の検証と評価を行います。

-   [Run](../run): セッションなどのジョブ設定オプションを使用してハードウェア上で実行します。

-   **APIリファレンス**: [Qiskit](/api/qiskit), [Qiskit Runtime IBM Client](/api/qiskit-ibm-runtime), 上記のAPIリファレンスドロップダウンメニューの[Qiskit IBM Runtime Rest API](/api/runtime/tags/programs)、[Qiskit IBM Provider](/api/qiskit-ibm-provider) format@@0(../errors) もここにあります。
