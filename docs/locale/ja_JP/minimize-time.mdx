# ジョブの実行時間を最小化する

処理とジョブの実行に費やす量子時間を制限する方法はいくつかあります。

-   必要な数だけ繰り返しとショットを実行します。作業負荷がかかる時間 (そのため) そのコスト)は、セッションで作成するジョブの数と、各ジョブで実行されるショットの数によって異なります。 したがって、必要なだけ多くのジョブやショットを実行することで、コストを管理することができます。

-   実行時間に制限を設定します。各ジョブまたはセッションの実行時間を制限できます。  詳細については、「Qiskit Runtime またはsessionの実行時間の最大値」(max-execution-time)を参照してください。

-   値が大きいほど、ジョブが長く実行される可能性があるため、エラー抑制、エラー軽減、および最適化に必要な設定のみを使用します。 詳細については、format@@0(advanced-runtime-options), format@@1(configure-runtime-compilation), format@@2(configure-error-軽減)を参照してください。
