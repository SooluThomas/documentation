# 利用可能なIBM Quantum シミュレーター

<Admonition type="note">
このページでは、IBM Quantum クラウドベースのシミュレータについて詳しく説明します。 Qiskit ビルトインシミュレーターの詳細については、[API リファレンスの Python ベースのシミュレーター] ページを参照してください。 api/qiskit/providers_basicaer) ローカルの状態ベクトルシミュレーションには、[Qiskit reference primitives](simulate-with-qiskit-primitives)を使用することもできます。
</Admonition>

IBM Quantumは、量子回路とアルゴリズムのプロトタイピングのための高性能シミュレータのコレクションを備えています。

利用可能なシミュレータを表示するには、画面の右上隅にあるアプリケーションスイッチャー(![switcher](/images/migration/switcher-small2)をクリックします。 ng) ), **リソースを計算** を選択して format@@0(https://quantum.ibm.com/services/resources?services=simulators) を表示し、すべてのシミュレータをクリックします。

以下のシミュレーション方法は、ジョブあたり最大300個の回路と8192個のショットをサポートしています。 以下に示す各シミュレータの詳細については、そのタイプ、説明、シミュレートする量子ビットの数などを参照してください。 ノイズモデリング、サポートされているゲートのリスト、および Qiskit Runtime を使用して呼び出す方法が含まれているかどうか。 `simulator_stateevector` は汎用ソリューションメソッドであるため、デフォルトでは良い選択です。 

<Admonition type="info" title="Simulator timeout limit">
  シミュレータが処理を妥当な時間で完了しないジョブを処理するのを防ぐため。 シミュレータに送信されるジョブは、10,000 秒以内の実行時間に制限されます(\~2)。 5時間)。
</Admonition>

## ステートベクトルシミュレーション

**タイプ:** Schro<unk> dinger wavefunction

**名前:** simulator_statevector

量子回路を、量子ビットの状態ベクトルの波動係数をゲートと命令が適用されるように計算します。 一般的なノイズモデリングに対応しています。

**Qubits:** 32

**ノイズモデリング:** はい

**サポートされているゲート/指示**

```python
['u1', 'u2', 'u3', 'u', 'p', 'r', 'rx', 'ry', 'rz', 'id',
'x', 'y', 'z', 'h', 's', 'sdg', 'sx', 't', 'tdg', 'swap',
'cx', 'cy', 'cz', 'csx', 'cp', 'cu1', 'cu2', 'cu3', 'rxx',
'ryy', 'rzz', 'rzx', 'ccx', 'cswap', 'mcx', 'mcy', 'mcz',
'mcsx', 'mcp', 'mcu1', 'mcu2', 'mcu3', 'mcrx', 'mcry',
'mcrz', 'mcr', 'mcswap', 'unitary', 'diagonal',
'multiplexer', 'initialize', 'kraus', 'roerror', 'delay']
```

**コード例**

```python
from qiskit_ibm_runtime import QiskitRuntimeService
service = QiskitRuntimeService()
backend = service.get_backend("simulator_statevector")
```

## スタビライザシミュレーション

**タイプ:** Clifford

**名前:** simulator_stabilizer

Clifford回路の効率的なシミュレータ。 ノイズオペレータがCliffordゲートでもある場合、ノイズの多い進化をシミュレートできます。

**Qubits:** 5000

**ノイズモデリング:** はい (Clifford only)

**サポートされているゲート/指示**

```python
['cx', 'cy', 'cz', 'id', 'x', 'y', 'z', 'h',
's', 'sdg', 'sx', 'swap', 'delay', 'roerror']
```

**コード例**

```python
from qiskit_ibm_runtime import QiskitRuntimeService
service = QiskitRuntimeService()
backend = service.get_backend("simulator_stabilizer")
```

## 拡張スタビライザーシミュレーション

**タイプ:** Clifford拡張（例：Clifford+T）

**名前:** simulator_extended_stablizer

ランク付けされたスタビライザー分解を使用して量子回路の動作を近似します。 Clifford以外のゲートの数がスタビライザの用語の数を決定します。

**Qubits:** 63

**ノイズモデリング:** いいえ

**サポートされているゲート/指示**

```python
['u0', 'u1', 'cx', 'cz', 'id', 'x', 'y', 'z', 'h',
't', 'tdg', 's', 'sdg', 'sdg', 'sx', 'swap', 'p', 'ccx', 'ccz',
'delay', 'roerror']
```

**コード例**

```python
from qiskit_ibm_runtime import QiskitRuntimeService
service = QiskitRuntimeService()
backend = service.get_backend("simulator_extended_stabilizer")
```

## MPSシミュレーション

**タイプ:** マトリクス製品の状態

**名前:** simulator_mps

状態にマトリックス製品状態(MPS)表現を使用するテンソルネットワークシミュレータ。 この表現は、エンタングルメントが弱い状態ではしばしばより効率的です。

**Qubits:** 100

**ノイズモデリング:** いいえ

**サポートされているゲート/指示**

```python
['unitary', 't', 'tdg', 'id', 'cp', 'u1', 'u2', 'u3', 'u', 'u',
'cx', 'cz', 'x', 'y', 'z', 'h', 'sdg', 'sx', 'swap',
'p', 'ccx', 'delay', 'roerror']
```

**コード例**

```python
from qiskit_ibm_runtime import QiskitRuntimeService
service = QiskitRuntimeService()
backend = service.get_backend("simulator_mps")
```

## QASMシミュレーション

**タイプ:** 一般的なコンテキスト対応

**名前:** ibmq_qasm_Simulator

理想的にはノイズモデリングを行う量子回路をシミュレートするための汎用シミュレータです。 シミュレーション方法は、入力回路とパラメータに基づいて自動的に選択されます。

**Qubits:** 32

**ノイズモデリング:** はい

**サポートされているゲート/指示**

```python
['u1', 'u2', 'u3', 'u', 'p', 'r', 'rx', 'ry', 'rz', 'id',
'x', 'y', 'z', 'h', 's', 'sdg', 'sx', 't', 'tdg', 'swap',
'cx', 'cy', 'cz', 'csx', 'cp', 'cu1', 'cu2', 'cu3', 'rxx',
'ryy', 'rzz', 'rzx', 'ccx', 'cswap', 'mcx', 'mcy', 'mcz',
'mcsx', 'mcp', 'mcu1', 'mcu2', 'mcu3', 'mcrx', 'mcry',
'mcrz', 'mcr', 'mcswap', 'unitary', 'diagonal',
'multiplexer', 'initialize', 'kraus', 'roerror', 'delay']
```

**コード例**

```python
from qiskit_ibm_runtime import QiskitRuntimeService
service = QiskitRuntimeService()
backend = service.get_backend("ibmq_qasm_simulator")
```

 Qiskit組み込みシミュレータは、[PythonベースのシミュレータAPI](/api/qiskit/providers_basicaer) リファレンスを参照してください。
